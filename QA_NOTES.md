# QA Notes

ブラウザでトップページを確認し、Beyond Japanese のファーストビュー、ナビゲーション、ヒーローコピー、提供済み横長ロゴの表示、写真プレースホルダー、主要セクションの本文が表示されていることを確認した。

ロゴは CSS で `width` のみ指定し、`height: auto` と `object-fit: contain` により、提供画像の縦横比を歪めない実装にしている。

写真・イラスト・追加アイコンなどの生成素材は使用していない。写真が必要な箇所は、ユーザー提供写真に差し替えやすい `PhotoPlaceholder` コンポーネントとして実装している。

`pnpm build` を実行し、ビルドは成功した。Vite のチャンクサイズ警告は出ているが、静的サイトテンプレートの依存関係に由来する警告であり、ビルド失敗ではない。

最終調整として、Testimonials セクションのサンプル注記をユーザー向けコピーに置き換え、Final CTA の仮メールリンクもページ内リンクに変更した。
